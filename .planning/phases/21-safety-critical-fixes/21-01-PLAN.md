    **SAFE-01 — Paper mode early-return in execution agent:**

    In `agents/execution/main.py`, after the deduplication check block (after line 632 where `continue` ends the dedup path), add a paper mode skip block BEFORE the circuit breaker check (before line 649). The block must:

    1. Check `if is_paper:`
    2. Log `logger.debug("paper_mode_order_skipped", order_id=order_id, note="paper_simulator handles execution")`
    3. Add to dedup: `processed_order_ids[order_id] = None`
    4. Evict oldest if over capacity: `while len(processed_order_ids) > _DEDUP_SET_MAX: processed_order_ids.popitem(last=False)`
    5. Ack the message: `await consumer.ack(channel, "execution_agent", msg_id)`
    6. `continue` to skip all remaining processing (circuit breaker, market fetch, algo select, execute, fill publish, DB write)

    This ensures the execution agent is a complete no-op in paper mode — only the paper_simulator processes fills.
