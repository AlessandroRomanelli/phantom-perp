  <action>
    Implement `compute_strategy_metrics` in `libs/metrics/engine.py`:

    **Internal helper `_compute_metrics(source: str, instrument: str, round_trips: list[RoundTrip]) -> StrategyMetrics`:**

    1. **Win/Loss classification (net_pnl basis per D-09):**
       - wins = [rt for rt in round_trips if rt.net_pnl > Decimal("0")]
       - losses = [rt for rt in round_trips if rt.net_pnl <= Decimal("0")]
       - win_count = len(wins), loss_count = len(losses), trade_count = len(round_trips)
       - win_rate = win_count / trade_count (float, 0.0-1.0)

    2. **Expectancy (METR-01):**
       - avg_win = sum(rt.net_pnl for rt in wins) / win_count if wins else Decimal("0")
       - avg_loss = sum(abs(rt.net_pnl) for rt in losses) / loss_count if losses else Decimal("0")
       - expectancy = avg_win * Decimal(str(win_rate)) - avg_loss * Decimal(str(1.0 - win_rate))
       - Guard: if no wins, avg_win = Decimal("0"); if no losses, avg_loss = Decimal("0")

    3. **Profit Factor (METR-02):**
       - gross_profit = sum(rt.gross_pnl for rt in round_trips if rt.gross_pnl > Decimal("0"))
       - gross_loss = abs(sum(rt.gross_pnl for rt in round_trips if rt.gross_pnl < Decimal("0")))
       - profit_factor = float(gross_profit / gross_loss) if gross_loss > Decimal("0") else None
       - Breakeven trades (gross_pnl == 0) excluded from both sums

    4. **Drawdown (METR-03):**
       - Sort round_trips by closed_at (should already be sorted from build_round_trips, but defensive sort)
       - cumulative = Decimal("0"), peak = Decimal("0"), peak_time = round_trips[0].closed_at
       - max_dd_amount = Decimal("0"), max_dd_duration_hours = 0.0
       - For each rt: cumulative += rt.net_pnl
         - If cumulative > peak: peak = cumulative, peak_time = rt.closed_at
         - dd = peak - cumulative
         - If dd > max_dd_amount: max_dd_amount = dd, max_dd_duration_hours = (rt.closed_at - peak_time).total_seconds() / 3600
       - After loop: if cumulative < peak (still in drawdown per D-07):
         - still_dd_duration = (datetime.now(timezone.utc) - peak_time).total_seconds() / 3600
         - If still_dd_duration > max_dd_duration_hours: max_dd_duration_hours = still_dd_duration

    5. **Fee-adjusted aggregates (METR-04/D-09):**
       - total_gross_pnl = sum(rt.gross_pnl for rt in round_trips)
       - total_fees_usdc = sum(rt.total_fees for rt in round_trips)
       - total_net_pnl = sum(rt.net_pnl for rt in round_trips)

    **Public function `compute_strategy_metrics(fills, min_trades=10)`:**
    - Call build_round_trips(fills) to get dict[tuple[str,str], list[RoundTrip]]
    - For each (source, instrument), round_trips in result:
      - If len(round_trips) < min_trades: result[(source, instrument)] = None (per D-01/D-02)
      - Else: result[(source, instrument)] = _compute_metrics(source, instrument, round_trips)
    - Return dict
