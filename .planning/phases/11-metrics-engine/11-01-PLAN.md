<objective>
Build the foundational data structures and round-trip reconstruction pipeline for the metrics engine.

Purpose: The metrics engine needs to transform raw `AttributedFill` rows (one per exchange fill) into closed `RoundTrip` objects (one per completed trade) before any performance metrics can be computed. This plan implements VWAP aggregation (D-05), FIFO entry/exit pairing (D-03), open position exclusion (D-04), and per-round-trip fee-adjusted P&L (METR-04 foundation).   <action>
    Replace `NotImplementedError` stubs in `libs/metrics/engine.py` with full implementations:

    **vwap_aggregate(fills: list[AttributedFill]) -> OrderResult:**
    - Assert len(fills) > 0
    - total_value = sum(f.price * f.size for f in fills) -- Decimal arithmetic
    - total_size = sum(f.size for f in fills)
    - avg_price = total_value / total_size
    - total_fee = sum(f.fee_usdc for f in fills)
    - filled_at = max(f.filled_at for f in fills) -- latest fill timestamp
    - Return OrderResult with fills[0].order_id, fills[0].instrument, fills[0].primary_source, fills[0].side, avg_price, total_size, total_fee, filled_at

    **pair_round_trips(orders: list[OrderResult]) -> list[RoundTrip]:**
    - Use collections.deque as FIFO entry stack
    - Iterate orders chronologically (caller must sort)
    - If stack empty or order.side == stack[0].side: push onto stack (new entry or pyramid)
    - If order.side != stack[0].side: pop entry from left (FIFO), create RoundTrip
    - For RoundTrip P&L:
      - If entry.side == "BUY" (LONG): gross_pnl = (exit.avg_price - entry.avg_price) * min(entry.total_size, exit.total_size)
      - If entry.side == "SELL" (SHORT): gross_pnl = (entry.avg_price - exit.avg_price) * min(entry.total_size, exit.total_size)
      - size = min(entry.total_size, exit.total_size) -- handle size mismatches conservatively
      - total_fees = entry.total_fee + exit.total_fee
      - net_pnl = gross_pnl - total_fees
    - Remaining stack entries are open positions -- excluded per D-04
    - Return list[RoundTrip]

    **build_round_trips(fills: list[AttributedFill]) -> dict[tuple[str, str], list[RoundTrip]]:**
    - Group fills by (primary_source, instrument) using collections.defaultdict(list)
    - For each group: sub-group by order_id, vwap_aggregate each sub-group -> list[OrderResult]
    - Sort OrderResults by filled_at ascending
    - Call pair_round_trips(sorted_orders)
    - Store in result dict keyed by (source, instrument)
    - Only include keys where round_trips is non-empty
