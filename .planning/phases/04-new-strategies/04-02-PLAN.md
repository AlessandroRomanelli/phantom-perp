Create the orderbook imbalance (OBI) strategy as a new SignalStrategy subclass that emits short-horizon directional signals from time-weighted bid/ask depth imbalance.

Purpose: OBI fills a coverage gap for short-term directional pressure. When bid depth consistently outweighs ask depth (or vice versa), it signals directional intent from market participants. With a 1-hour time horizon, this captures opportunities the medium/long-horizon strategies miss.

Output: Complete OBI strategy with config, registration, strategy matrix entry, and comprehensive tests.

    1. `OrderbookImbalanceParams` dataclass:
    ```python
    @dataclass
    class OrderbookImbalanceParams:
        lookback_bars: int = 10           # 10 bars * 60s = 10 min window (OBI-02)
        imbalance_threshold: float = 0.25 # Minimum TWA imbalance to trigger
        max_spread_bps: float = 20.0      # Suppress if spread > 20 bps (OBI-03, thin book)
        atr_period: int = 14
        stop_loss_atr_mult: float = 1.5   # Tight stops for short horizon
        take_profit_atr_mult: float = 2.0
        min_conviction: float = 0.45      # Higher bar due to noisy data (D-10)
        cooldown_bars: int = 3            # Short cooldown for frequent firing (D-09)
        portfolio_a_min_conviction: float = 0.65  # Portfolio A threshold (OBI-04)
        enabled: bool = True
    ```

    2. `OrderbookImbalanceStrategy` class extending `SignalStrategy`:
    - Constructor: `__init__(self, instrument: str, params: OrderbookImbalanceParams | None = None, config: dict[str, Any] | None = None)`
    - Load params from config dict if provided (same pattern as CorrelationStrategy)
    - Track `_bars_since_signal: int` for cooldown
    - `min_history` property returns `lookback_bars + atr_period`

    3. `evaluate()` method:
    a. Increment `_bars_since_signal`, check cooldown
    b. Check `store.sample_count >= self.min_history`
    c. Get `imbalances = store.orderbook_imbalances`
    d. Compute time-weighted imbalance using linear weights over last `lookback_bars` samples:
       ```python
       window = imbalances[-p.lookback_bars:]
       weights = np.arange(1, len(window) + 1, dtype=np.float64)
       tw_imbalance = float(np.average(window, weights=weights))
       ```
    e. Check depth gate: if `snapshot.spread_bps > p.max_spread_bps`, return []
    f. Check threshold: if `abs(tw_imbalance) < p.imbalance_threshold`, return []
    g. Determine direction: LONG if tw_imbalance > 0, SHORT if tw_imbalance < 0
    h. Compute ATR for stops: `atr_vals = atr(highs, lows, closes, p.atr_period)`
    i. Compute 3-component conviction:
       - Imbalance magnitude (0-0.45): `imb_excess = abs(tw_imbalance) - p.imbalance_threshold; imb_score = min(max(imb_excess / p.imbalance_threshold * 0.45, 0.0), 0.45)`
       - Spread quality (0-0.30): `spread_score = max(0.0, min((20.0 - snapshot.spread_bps) / 20.0 * 0.30, 0.30))`
       - Volume component (0-0.25): compute volume ratio = latest volume / mean volume over lookback. `vol_score = min(max((volume_ratio - 0.5) / 2.0, 0.0), 0.25)`
       - `conviction = round(min(imb_score + spread_score + vol_score, 1.0), 3)`
    j. Check min_conviction
    k. Portfolio routing: `suggested_target = PortfolioTarget.A if conviction >= p.portfolio_a_min_conviction else PortfolioTarget.B`
    l. Compute entry, stop, TP using ATR (same pattern as correlation: `round_to_tick(entry +/- atr_d * mult)`)
    m. Build StandardSignal with source=SignalSource.ORDERBOOK_IMBALANCE, time_horizon=timedelta(hours=1), metadata including tw_imbalance, spread_bps
    n. Reset `_bars_since_signal = 0`
