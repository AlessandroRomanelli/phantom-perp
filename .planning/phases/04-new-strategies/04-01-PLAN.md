    Create `agents/signals/funding_filter.py` with:

    1. `FundingBoostResult` frozen dataclass with fields: `boost: float`, `z_score: float`, `decay_factor: float`, `aligned: bool`

    2. `compute_funding_boost()` function with signature:
    ```python
    def compute_funding_boost(
        funding_rates: NDArray[np.float64],
        signal_direction: PositionSide,
        hours_since_last_funding: float,
        z_score_threshold: float = 1.5,
        max_boost: float = 0.10,
        lookback: int = 50,
        min_samples: int = 10,
    ) -> FundingBoostResult:
    ```

    Implementation logic:
    a. Guard: if len(funding_rates) < min_samples, return FundingBoostResult(boost=0.0, z_score=0.0, decay_factor=0.0, aligned=False)
    b. Compute z-score: window = funding_rates[-lookback:], mean = np.mean(window), std = np.std(window, ddof=1), if std < 1e-12 return z_score=0.0, else z_score = (funding_rates[-1] - mean) / std
    c. Check alignment: LONG aligned when funding_rates[-1] < 0 (negative funding = bullish), SHORT aligned when funding_rates[-1] > 0 (positive funding = bearish). If not aligned, return boost=0.0.
    d. Compute settlement decay: decay_factor = exp(-2.0 * (1.0 - hours_since_last_funding)). Clamp hours_since_last_funding to [0, 1].
    e. Compute boost: if abs(z_score) >= z_score_threshold and aligned: boost = min(max_boost * (abs(z_score) / z_score_threshold) * decay_factor, max_boost). Else boost = 0.0.
    f. Return FundingBoostResult(boost=round(boost, 4), z_score=round(z_score, 4), decay_factor=round(decay_factor, 4), aligned=aligned)

    Imports: `from __future__ import annotations`, `from dataclasses import dataclass`, `import numpy as np`, `from numpy.typing import NDArray`, `from libs.common.models.enums import PositionSide`

    Create `agents/signals/tests/test_funding_filter.py` with tests matching the behavior spec above. Use direct numpy arrays as input (no FeatureStore needed). Follow the test pattern from test_correlation.py.
